# Text Converter App

This is a simple React project that allows users to convert text to:
## Features
- Uppercase
- Lowercase
- Remove extra spaces
- Convert to sentence case 
- Count words and characters
- Clear text  

## Features
- Live text editing
- Dark/light mode toggle
- Instant conversion results

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`

## Technologies Used
- React.js
- Bootstrap
- JavaScript
